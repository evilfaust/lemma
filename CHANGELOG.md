# Changelog — История изменений

## [3.3.1] - 2026-02-15

### Изменено
- **StudentProgressDashboard** — в модальном окне «Детали» удалено редактирование класса ученика; класс теперь только для просмотра (управление классами перенесено в админку PocketBase).
- **StudentApp (AchievementGallery)** — галерея достижений разделяет прогресс:
  - достижения за текущую сессию (текущий тест),
  - достижения за все попытки ученика.
- **Легенда статусов достижений** — добавлены маркеры:
  - зелёная точка — получено за текущий тест,
  - синяя точка — получено ранее,
  - замок — не получено.
- **UI достижений** — выровнены размеры карточек в сетке, маркеры перенесены в угол карточки, улучшена читаемость маркеров.
- **StudentTestPage / StudentResultPage** — в шапке показывается название работы (`work.title`) вместо заголовка для учеников (`student_title`).

---

## [3.3.0] - 2026-02-14

### Добавлено
- **Глубокий редизайн модуля «Теория»** — полная переработка всех 5 компонентов
  - **TheoryBrowser** — hero-секция со статистикой, градиентные карточки, категории-pills с цветными точками, кастомные карточки статей с hover-анимациями
  - **TheoryEditor** — rich formatting toolbar (заголовки, форматирование, списки, таблицы, формулы, изображения, ссылки), split-pane с ресайзом мышкой, status bar (слова, формулы, символы, тема, автосохранение)
  - **TheoryArticleView** — TOC sidebar с IntersectionObserver, floating action bar (Edit/Print/PDF/Theme), связанные задачи горизонтальными карточками
  - **TheoryCategoryManager** — визуальные карточки с градиентными иконками вместо таблицы, drag-and-drop для порядка
  - **TheoryPrintBuilder** — sidebar с выбором и drag-and-drop переупорядочиванием статей, индикатор кол-ва страниц
- **EditorToolbar** — новый субкомпонент с 4 группами кнопок форматирования для Monaco Editor
- **TableInsertPopover** — popover для вставки таблиц NxM
- **5 тем оформления** — classic, minimal, academic, notebook, compact (ранее только 2)

### Изменено
- Full-height layout для theory-view и theory-print (убраны лишние padding/margin)
- 7 новых CSS-файлов для компонентов теории
- Все компоненты используют staggered-анимации, HTML5 Drag API, современные карточки

## [3.1.0] - 2026-02-12

### Добавлено
- **Система достижений** — 30 предустановленных достижений с анимированными иконками
  - Случайные бейджи (common, rare, legendary) — один за попытку
  - Условные достижения (score, speed, count, special) — автоматическая разблокировка
  - Галерея достижений для ученика
  - Ручная выдача достижений учителем
  - Включение/отключение на уровне сессии
  - Polling каждые 10 секунд для обновления наград от учителя
- **Авторизация учеников** — регистрация (имя + логин + пароль), авторизация через PocketBase Auth
  - Привязка гостевых попыток к аккаунту
  - Повторное прохождение тестов под контролем учителя
- **Панель прогресса учеников** (StudentProgressDashboard) — таблица всех зарегистрированных учеников с попытками, средним баллом, классом
- **Список выполненных работ ученика** с deep-link на редактор работ
- **Модальное окно достижений учителя** с просмотром уже выданных бейджей

### Изменено
- Улучшен UI студенческого приложения (StudentApp)
- Исправлена синхронизация потока ученика (attempt reuse, answers persistence)
- Исправлена работа парсера/вывода, ручных достижений учителя
- Усилен скрипт бэкапа
- Убран серый фон у achievement-lock-overlay и иконок достижений

## [3.0.0] - 2026-02-10

### Добавлено
- **Студенческий интерфейс** — полный поток тестирования
  - StudentAuthPage — авторизация/регистрация
  - StudentEntryPage — стартовый экран теста
  - StudentTestPage — прохождение теста с автосохранением ответов
  - StudentResultPage — просмотр результатов
  - Dual routing: `/student/{sessionId}` → StudentApp
- **Редактор работ** (WorkEditor + WorkEditorPage) — управление вариантами и задачами сохранённых работ
  - Добавление/удаление/перетаскивание задач
  - Замена и редактирование задач
  - Панель сессии (SessionPanel) — выдача, QR-код, ссылка
  - Результаты учеников (TeacherResultsDashboard)
  - Перемещение задач между вариантами
- **Менеджер работ** (WorkManager) — список работ с поиском, фильтрами, архивированием
- **Проверка ответов** (answerChecker.js) — числовое сравнение, дроби, LaTeX, альтернативы через `|`
- **Сессии тестирования** (work_sessions) — выдача работ ученикам с QR-кодом
- **Попытки** (attempts) — отслеживание статуса, оценок, длительности
- **Ответы по задачам** (attempt_answers) — хранение и проверка ответов
- **Коллекция students** — Auth-коллекция для учеников
- **useStudentSession** — хук для управления сессией ученика
- **ReferenceDataContext** — глобальный контекст справочных данных (topics, tags, subtopics, years, sources)
- **Парсинг sdamgia.ru** — эндпоинт POST /parse-sdamgia в PDF-сервисе
  - Конвертация HTML-таблиц в Markdown
  - Очистка LaTeX-формул из русскоязычных alt-текстов
- **Заголовок для учеников** (student_title) в сессии

### Изменено
- Рефакторинг TaskCatalogManager — декомпозиция на подкомпоненты (TopicTab, SubtopicTab, TagTab, SourceTab, OtherTab, DuplicateTab, MergeModal)
- Рефакторинг: prop drilling заменён на ReferenceDataContext
- Ленивая загрузка TheoryEditor для уменьшения начального бандла
- Исправлено экранирование строковых значений в фильтрах PocketBase
- Заменена biased сортировка на Fisher-Yates shuffle
