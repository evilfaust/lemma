# Changelog - История изменений

## [2.1.0] - 2026-02-07

### 🎉 Добавлено
- **Массовые операции с задачами**: теги (добавить/заменить), сложность, источник, тема, подтемы, удаление
- **Редактирование сохранённых работ** (update work/variants) в генераторах
- **Автопрогрессия сложности** в новых генераторах
- **Дашборд статистики** по темам, подтемам, тегам, сложности и источникам
- **Облако тегов** со ссылкой на отфильтрованные задачи
- **Связь теория → задачи** по тегам статьи
- **Каталог справочников** с CRUD, объединением дублей и переходом к задачам
- **Проверка дублей** по темам/подтемам/тегам/источникам/условиям задач
- **Скрипты обслуживания**: `fix_task2_tables.py`, `recolor_tags.py`

### 🔄 Изменено
- **TaskFilters / TaskList**: поддержка внешнего применения фильтров (переход из статистики)
- **useWorksheetGeneration**: прогрессивная сложность как опция генерации
- **useWorksheetActions**: обновление сохранённых работ
- **TaskFilters**: фильтр по наличию изображения

## [2.0.0] - 2026-02-04

### 🎉 Добавлено

#### Генератор контрольных работ
- **Новый компонент `TestWorkGenerator.jsx`** - генератор контрольных работ с задачами из разных тем
  - Конструктор структуры работы с блоками задач
  - Каждый блок настраивается независимо (тема, подтемы, сложность, теги, количество)
  - Предпросмотр структуры с временной шкалой
  - Статистика: блоки, темы, задачи

- **Три режима генерации вариантов**:
  - Разные задачи в каждом варианте (без повторений)
  - Одинаковые задачи, разный порядок (перемешивание)
  - Одинаковые задачи, одинаковый порядок

- **Гибкие фильтры для каждого блока**:
  - Тема (обязательный)
  - Подтемы (multiple, с автообновлением при смене темы)
  - Уровни сложности (multiple: 1-5)
  - Теги (multiple)
  - Источник
  - Год

- **Все возможности форматирования**:
  - Компактный режим
  - Настройка шрифта (10pt-20pt)
  - Количество колонок (1-3)
  - Место для решения (нет/мало/средне/много)
  - Фильтр префиксов задач
  - Поля для ФИО
  - Лист с ответами

- **Drag & Drop** - перетаскивание задач между вариантами
- **Замена и редактирование задач** прямо в варианте
- **Сохранение/загрузка работ** в базу данных
- **Экспорт в PDF** с сохранением всех настроек

#### Архитектурные улучшения

**Новые переиспользуемые хуки** (`src/hooks/`):

1. **`useWorksheetGeneration.js`** - универсальная логика генерации вариантов
   - Поддержка структур блоков с задачами из разных тем
   - Автоматическое исключение дубликатов
   - Три режима генерации
   - Сортировка: по коду, сложности, случайная
   - Умная обработка фильтров

2. **`useTaskDragDrop.js`** - drag & drop функционал
   - Перетаскивание задач внутри варианта и между вариантами
   - Визуальная индикация (dragging, drag-over)
   - События: dragStart, dragOver, dragLeave, drop, dragEnd

3. **`useWorksheetActions.js`** - действия с работами
   - Печать через window.print()
   - Экспорт в PDF (html2pdf.js)
   - Сохранение работ в БД
   - Загрузка сохранённых работ
   - Удаление работ

**Новые переиспользуемые UI компоненты** (`src/components/worksheet/`):

1. **`FilterBlock.jsx`** - блок фильтров для подбора задач
   - Универсальный компонент для всех генераторов
   - Автообновление подтем при смене темы
   - Multiple select для подтем, сложности, тегов
   - Опциональные фильтры (источник, год)

2. **`VariantSettings.jsx`** - настройки генерации вариантов
   - Количество вариантов (1-50)
   - Режим генерации (разные/перемешанные/одинаковые)
   - Тип сортировки
   - Статистика общего количества задач

3. **`FormatSettings.jsx`** - настройки форматирования
   - Колонки (1-3)
   - Размер шрифта (10pt-20pt)
   - Место для решения
   - Компактный режим
   - Фильтр префиксов
   - Настройки отображения ответов
