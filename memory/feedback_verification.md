name: Проверка изменений — только сборка
description: Не запускать dev-сервер для проверки изменений, достаточно npm run build
type: feedback
---

Не запускать dev-сервер (preview_start) для проверки изменений фронтенда. Достаточно выполнить `cd ege-tasks && npm run build`.

**Why:** Пользователь явно указал, что запуск dev-сервера для проверки не нужен.

**How to apply:** После редактирования React-компонентов запускать только `npm run build` для проверки корректности. Не вызывать preview_start, preview_screenshot и другие preview_* инструменты.
