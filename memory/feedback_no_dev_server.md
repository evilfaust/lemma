name: Не запускать dev-сервер для проверки
description: Пользователь проверяет изменения сам, не нужно запускать preview/dev-сервер
type: feedback
---

Не запускать dev-сервер (preview_start, npm run dev) для верификации изменений.

**Why:** Пользователь явно попросил не делать проверку через запуск dev-сервера.

**How to apply:** После редактирования кода не вызывать preview_start и не пытаться верифицировать через preview_*. Достаточно убедиться что сборка (`npm run build`) проходит без ошибок.
