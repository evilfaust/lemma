# Raspberry Pi Deployment

Скрипты и конфигурации для развертывания EGE Tasks Manager на Raspberry Pi.

## 📁 Файлы

- **`deploy-to-raspberry.sh`** — Основной скрипт деплоя (запускается на Mac)
- **`copy-configs-to-raspberry.sh`** — Копирование конфигов на малину перед setup
- **`setup-raspberry.sh`** — Первичная настройка Raspberry Pi (запускается на RPi)
- **`nginx.conf`** — Конфигурация Nginx
- **`pocketbase.service`** — Systemd сервис для PocketBase
- **`pdf-service.service`** — Systemd сервис для PDF-генератора

## 🚀 Быстрый старт

### 1. Первоначальная настройка (один раз)

```bash
# На Mac: копируем конфиги на Raspberry Pi
./raspberry/copy-configs-to-raspberry.sh

# На Raspberry Pi: запускаем первичную настройку
ssh faust@192.168.1.68
cd /home/faust/ege-app-setup
chmod +x setup-raspberry.sh
./setup-raspberry.sh
exit
```

### 2. Деплой приложения (при каждом обновлении)

```bash
# На Mac: запускаем деплой
./raspberry/deploy-to-raspberry.sh
```

После деплоя приложение доступно по адресу: `http://192.168.1.68`

## 📱 Telegram-бот

При включении Raspberry Pi бот автоматически отправляет уведомление с:
- 🌐 IP-адресом в текущей сети
- 📡 Именем Wi-Fi сети
- ✅ Статусом всех сервисов
- 🔗 Прямой ссылкой на приложение

**Команды бота:** `/status`, `/ip`, `/stats`, `/restart`, `/notify`

Подробности в **TELEGRAM_BOT.md**

## 📡 Wi-Fi сети

Raspberry Pi настроен на автоматическое подключение к следующим сетям:
- **VZMAKH_5G** (приоритет 20) — школьная сеть 5 GHz
- **VZMAKH_2G** (приоритет 10) — школьная сеть 2.4 GHz
- **Keenetic-1965** (приоритет 0) — домашняя сеть

Подробности в **WIFI_SETUP.md**

## ⚙️ Оптимизация

Графический интерфейс **отключен** для экономии ресурсов:
- Освобождается ~100-150 MB RAM
- Быстрее загрузка системы
- Больше ресурсов для приложения

Применится после перезагрузки Raspberry Pi.

## 🔧 Требования

**На Mac:**
- rsync
- ssh/scp
- Node.js (для сборки фронтенда)

**На Raspberry Pi:**
- Raspberry Pi OS (64-bit рекомендуется)
- Node.js 18+ (устанавливается автоматически)
- Nginx (устанавливается автоматически)
- 2GB+ RAM (рекомендуется)
- 4GB+ свободного места на диске

## ⚠️ Важно

- Скрипт деплоя **полностью заменяет** БД на Raspberry Pi данными с Mac
- Перед каждым деплоем создается бэкап БД (на Mac и на RPi)
- Основной режим работы (dev на Mac) остается неизменным
- Raspberry Pi — это дополнительный production-сервер для стабильных версий
