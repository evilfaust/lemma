# План: Объединение генераторов в один компонент

## Суть проблемы
- WorksheetGenerator + PrintableWorksheet — устаревший генератор карточек с фиксированной A4 сеткой, пустым пространством, лишними рамками
- TaskSheetGenerator (OralWorksheetGenerator) — более продвинутый генератор с drag-and-drop, заменой, редактированием, компактным режимом

## Решение
Удалить WorksheetGenerator и PrintableWorksheet. Добавить в TaskSheetGenerator функционал сохранения/загрузки карточек из коллекции `cards`.

## Шаги

### 1. Добавить в TaskSheetGenerator сохранение в cards
- Кнопка «Сохранить карточки» рядом с существующими кнопками
- При нажатии: каждый вариант сохраняется как отдельная карточка в коллекции `cards`
  - `title`: `{название} {номер_варианта_с_нулями}` (например "Логарифмы 001")
  - `tasks`: массив ID задач варианта
  - `format`: формат (сохраняем для справки)
  - `show_answers`, `show_solutions`: текущие настройки
  - `note`: опционально
- API функции `createCard`, `getCards`, `getCard`, `deleteCard` уже есть в pocketbase.js

### 2. Добавить панель «Сохранённые карточки» в TaskSheetGenerator
- Новая панель Collapse «Сохранённые карточки» с таблицей
- Колонки: Название, Заданий, Дата, Действия (Открыть / Удалить)
- «Открыть» загружает карточку как один вариант (аналогично handleLoadCard из WorksheetGenerator)
- «Удалить» удаляет карточку с подтверждением
- Таблица загружается при монтировании компонента

### 3. Убрать сохранение через works/variants
- Удалить кнопку «Сохранить работу» и модальное окно saveModal
- Удалить кнопку «Открыть сохраненную» и модальное окно loadModal
- Удалить связанные состояния и обработчики: saveModalVisible, savingWork, loadModalVisible, savedWorks, loadingWorks
- Удалить функции: handleSaveWork, handleOpenLoadModal, handleLoadWork, handleDeleteWork

### 4. Обновить App.jsx
- Удалить импорт WorksheetGenerator
- Удалить пункт меню 'generator'
- Оставить только два пункта: «Все задачи» и «Листы задач» (или переименовать в «Генератор карточек»)
- Удалить case 'generator' из renderContent и getHeaderTitle
- Удалить мёртвый закомментированный код (строки 187-329)

### 5. Удалить файлы
- Удалить `WorksheetGenerator.jsx`
- Удалить `PrintableWorksheet.jsx`

### 6. Проверить сборку
- `npm run build` для проверки что всё компилируется

## Что НЕ меняется
- Вся логика генерации вариантов (по тегам, по сложности, стандартная)
- Drag-and-drop перестановка задач
- Замена и редактирование задач
- Все настройки формата печати (колонки, шрифт, компактный режим и т.д.)
- PDF экспорт
- Фильтры задач
